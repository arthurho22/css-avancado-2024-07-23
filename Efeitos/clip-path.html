clip-path

O clip-path é uma propriedade css que permite criar formas complexas cortando elementos de acordo com um caminho específico. É util para criar layouts e efeitos visuais interessantes.

Como funciona o clip-path

o clip-path define uma área visível de um elemento, escondendo o que está fora dessa área. Existem várias formas que você pode usar com clip-path, como:

  • polygon(): Define uma forma usando uma lista de coordenadas que formam um polígono.
  • circle(): Define uma forma circula.
  • ellipse(): Define formas elípticas.
  • inset(): Define um retângulo interno com uma margem opcional.
